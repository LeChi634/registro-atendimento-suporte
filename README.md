# Registro de Atendimento de Suporte em TI

Projeto desenvolvido em Python com o objetivo de simular o registro básico de atendimentos de suporte técnico, semelhante ao que é realizado em rotinas de Help Desk e Suporte de TI.

## Funcionalidades
- Menu interativo com categorias de problemas de TI
- Validação de entrada do usuário
- Coleta de dados do atendimento (usuário, setor e descrição)
- Registro de data e hora do atendimento
- Geração automática de log em arquivo `.txt`

## Tecnologias Utilizadas
- Python 3
- Execução via terminal (VS Code)

## Como Executar
1. Certifique-se de ter o Python 3 instalado
2. Clone este repositório ou baixe os arquivos
3. No terminal, execute:

```bash
python registro_suporte.py
